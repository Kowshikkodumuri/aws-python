WHAT IS GIT?
# Git is a version control system.

It helps you:
--------------
# Track changes in your code.
# Collaborate with others without messing up the codebase.
# Revert to previous versions if something breaks.
# Think of it like a time machine for your code.

Key Features:
--------------
# Tracks every code change.
# Allows branching and merging.
# Works locally (no internet required).


WHAT IS GITHUB?
# GitHub is a cloud-based platform that hosts Git repositories.
# It lets you share your Git-managed code online.
# Helps in collaboration, especially in open-source and team projects.

Key Features:
---------------
# Online code hosting (for Git repos).
# Collaboration tools (issues, pull requests, reviews).
# Integration with CI/CD, deployment, and more.
